LINGUAGEM HTML Significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), juntamente com CSS e Javascript, é um componente básico da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos. Atualmente a linguagem de marcação HTML encontra-se na versão 5 e é padronizada pelo W3C (World Wide Web Consortium), uma organização internacional responsável por estabelecer padrões para a internet. COMO EXECUTAR O HTML Para executar o código HTML da página da Web (conhecido como código-fonte), primeiramente abra um navegador da Web e pesquise o local do seu arquivo, ou simplesmente execute o arquivo a partir de sua localizalização no Sistema Operacional.
